## Architecture Design
- Create a P2P Conn-pool
- BlockChain Generate
- Node Manage And Vote
- Pick Node
- Write Block On Blockchain

## RUN 
```
git clone git@github.com:csunny/dpos.git

cd dpos    
go build main/dpos.go
```

connect multi peer 
```
./dpos new --port 3000 --secio
```
